# CRM Architecture 2.0

# Проект «У Тимоши»

Версия: 1.0

Статус: утверждено

---

# 1. Назначение CRM

CRM является центральной системой проекта.

Все данные клиентов, автоматов, покупок, бонусов и программ лояльности хранятся в CRM.

CRM взаимодействует с:

* вендинговыми автоматами;
* Telegram Bot;
* Telegram Mini App;
* сайтом;
* платёжной системой;
* административной панелью.

---

# 2. Основные сущности

## Customer

Карточка клиента.

Поля:

* ID
* Имя
* Телефон
* Email
* Дата рождения
* Telegram ID
* Telegram Username
* VK ID
* Дата регистрации
* Статус
* Доверенный клиент

---

## Club Account

Клуб Тимоши.

Поля:

* Customer ID
* Баланс клуба
* Дата вступления
* Статус активности
* Размер скидки

---

## Bonus Account

Бонусный счёт.

Поля:

* Customer ID
* Баланс бонусов
* Всего начислено
* Всего списано

---

## Bonus Transaction

История бонусов.

Поля:

* ID
* Customer ID
* Тип операции
* Сумма
* Дата
* Комментарий

# 7. Юридический блок

## Consent

Хранение согласий клиента.

Типы:

* обработка персональных данных;
* маркетинговые материалы;
* участие в фото-челлендже;
* Telegram уведомления;
* Email рассылка.

Поля:

* Customer ID
* Тип согласия
* Версия документа
* Дата принятия
* IP адрес
* Источник

---

# 8. Доверенный клиент

Статус присваивается автоматически.

Условия:

* указан Email;
* подписка на Telegram канал;
* подписка на VK;
* согласие на рекламу;
* участие в Клубе Тимоши.

---

# 9. День рождения

Условия получения подарка:

* дата рождения заполнена;
* баланс клуба не менее 300 ₽;
* статус клуба активен.

Награда:

1 бесплатная порция мороженого.
