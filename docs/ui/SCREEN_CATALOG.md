# Admin Console Screen Catalog v1

Status: Specification
Version: 1.0
Date: 2026-07-23

## 1. How to read this catalog

Every planned v1 screen is listed below. Each entry defines purpose, displayed data, actions, filters, sorting, search, bulk actions, permissions, related modules and future extensions. `None` is an intentional design decision. All permissions are additionally limited by organization/region/location/machine scope and owning-domain policy.

Common behavior from `ADMIN_CONSOLE_UI_SPEC.md` applies to every screen.
