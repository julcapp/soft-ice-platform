# Telegram Bot — ice_robo_Bot

Минимальный Telegram-бот для проверки ЮKassa и дальнейшего развития Soft ICE Platform.

## Назначение

Бот показывает:

- приветственное меню;
- карточку мороженого «Вкус дня» — 130 ₽;
- карточку «Клуб Тимоши» — 300 ₽;
- карточку «Подарочный сертификат» — 500 ₽;
- переход в Telegram Mini App по адресу `https://app.utimoshi.ru`.

## Установка

```bash
cd telegram-bot
npm install
cp .env.example .env
```

В файле `.env` указать токен из BotFather:

```env
BOT_TOKEN=0000000000:replace_me
MINI_APP_URL=https://app.utimoshi.ru
WEBSITE_URL=https://utimoshi.ru
BOT_DISPLAY_NAME=ice_robo_Bot
```

## Запуск

```bash
npm start
```

Для разработки:

```bash
npm run dev
```

## Проверка сценария для ЮKassa

1. Открыть `ice_robo_Bot` / `@desserty_bot`.
2. Сделать полноэкранный скриншот стартового экрана.
3. Нажать `START`.
4. Сделать скриншот приветствия и меню.
5. Открыть карточку `🍦 Купить мороженое`.
6. Сделать скриншот с ценой `130 ₽`.
7. Открыть карточку `🎁 Клуб Тимоши`.
8. Сделать скриншот с ценой `300 ₽`.
9. Открыть карточку `🎫 Подарочный сертификат`.
10. Сделать скриншот с ценой `500 ₽`.
11. Нажать `📱 Открыть Mini App`.
12. Сделать скриншот перехода и открытого Mini App.

## Важно

Скриншоты для ЮKassa должны быть отдельными полноэкранными файлами, не коллажем.
